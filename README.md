# OS-CA-IPC-Debugger

This repository contains a frontend application for simulating and visualizing Inter-Process Communication (IPC) mechanisms, such as Pipes, Queues, and Memory Management.

## Repository Structure

- `frontend/` - Vite + React frontend for IPC simulation and visualization
- `.gitignore` - Root ignore file for node_modules, build outputs, and environment files
- `README.md` - Project documentation

## Getting Started

See `frontend/README.md` for setup and usage instructions for the frontend application.
